# 📊 Trader Behavior Analysis using Market Sentiment

## 📌 Objective

This project explores the relationship between Bitcoin market sentiment (Fear/Greed) and trader performance using historical trading data.

## 📂 Datasets Used

* Bitcoin Fear & Greed Index
* Historical Trader Data (Hyperliquid)

## ⚙️ Approach

1. Data Cleaning and preprocessing
2. Merging datasets on date
3. Exploratory Data Analysis (EDA)
4. Visualization of key metrics
5. Insight extraction

## 📊 Key Findings

* Traders achieve highest profitability during **Extreme Greed**, indicating strong bullish trends improve performance.
* Win rates are higher in bullish conditions, suggesting momentum-driven success.
* Larger trade sizes are observed during **Fear**, indicating risk-taking or recovery attempts.
* Selling activity increases during Greed phases, suggesting profit-booking behavior.
* Most trades yield small returns, while a few large trades dominate overall profitability.

## 📈 Visualizations

* Average PnL by sentiment
* Win rate comparison
* Trade size distribution
* Buy vs Sell behavior
* 3D visualization (Trade Size vs PnL vs Sentiment)

## 🧠 Conclusion

Market sentiment plays a significant role in shaping trader behavior and performance. Traders tend to perform better during bullish conditions, while risk-taking behavior increases during uncertain or fearful markets.
